1. Open the `index.html` file in a web browser.
2. Log in using the credentials:
   - Username: `user`
   - Password: `password123`
3. Upon successful login, you will be redirected to the `home.html` page.
4. Click the "Logout" button on the home page to log out and return to the login page.

This application uses localStorage to manage the login state. For a production environment, implement a secure backend service for authentication.
